Quick Setup Guide

1. Register a Publisher

Go to the Publishers tab and add a new publisher with their site ID and domain.

2. Create Ads

Go to the Ads tab and create ad campaigns. Assign them to positions and publisher sites.

3. Integration Script

Publishers add this to their website:

<div class="adnet-slot" data-position="header"></div>
<div class="adnet-slot" data-position="sidebar"></div>
<div class="adnet-slot" data-position="inline"></div>

<script src="https://indranexadnet.dpdns.org/ad.js" data-site-id="SITE_ID"></script>

4. Available Positions

header, footer, sidebar, inline, custom (any string)
